I built a Selenium automation framework using Python and Pytest. 
The framework follows the Page Object Model design pattern to separate UI locators and actions from test logic. 
Browser setup and teardown are handled using Pytest fixtures in conftest.py. 
The framework supports data-driven testing using pytest parameterization and parallel execution using pytest-xdist. 
Test data and configuration are maintained in the utils folder, making the framework modular, maintainable, and scalable.
